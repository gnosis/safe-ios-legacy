ResultSet

public protocol ResultSet : AnyObject

Represents one row of the query result. You can move to the next row with ResultSet.advanceToNextRow() method and fetch column values of the current row as String, Int, Double, or Data.

  • advanceToNextRow()

    Moves to the next result row, returning true when there is more to get, and false if that is the last result row.

    Throws

    Throws error if there was a problem in the database.

    Declaration

    Swift

    func advanceToNextRow() throws -> Bool

    Return Value

    true if more rows available, false if no more rows available.
