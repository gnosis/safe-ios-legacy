  • execute(sql:bindings:dict:) Default implementation

    Executes arbitrary SQL statement, binding values from bindings and dict arguments to it. This method is meant for update queries.

    See execute(sql:bindings:dict:resultMap:) for executing SQL select queries that return result.

    Throws

    may throw error if there is a problem with SQL statement or with the database.

    bindings

    dict
